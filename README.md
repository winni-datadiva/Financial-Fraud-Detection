## **Financial Fraud Detection**
<br/> Financial Fraud Detetection modeling project for data science fellowship </br>
<br/>
##### Project Instruction: You're a data scientist working for a NYC-based international bank called Caishen. The company announced in an all-hands meeting that the goal for the next 3 years will be to identify fraudulent activity within their customer-facing bank accounts. While your cybersecurity team has provided you with a dataset of historic fraudulent activity, the responsibility of designing a minimal-viable-product (MVP) has fallen on your desk.  </br>
<br/>
*Project Objectives:* create an **ensemble classifier (random forest or boosted model)** that will assess if fraudulent activity has occurred for a transaction. Including, but not limited to: </br>
- Initial EDA </br>
- Data cleaning, wrangling, & pre-processing </br>
- Model creating, hyperparameter search, & model evaluation </br>
- Generate report </br>
